Objetivo do sistema
O sistema tem como objetivo simular uma biblioteca pessoal de livros que estão sendo lidos, sendo útil para a organização e para manter o usuário atualizado em relação à sua leitura.

Funcionalidades implementadas:
Adicionar livros (Capa, título, autor ,pagina, gênero).
Editar livro.
Excluir livro.
Listar livro.
Adicionar genero ( genero,descrição).
Excluir gênero;
Editar gênero;
Listar gênero;

Estrutura das tabelas
CREATE TABLE tb_generos (
    id INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(120) NOT NULL,
    descricao TEXT
);

CREATE TABLE tb_livros(
    id INT UNSIGNED NOT NULL AUTO_INCREMENT PRIMARY KEY,
    genero_id INT UNSIGNED,
    capa VARCHAR(255),
    titulo VARCHAR(100) NOT NULL,
    autor VARCHAR(100) NOT NULL,
    paginas INT NOT NULL DEFAULT 0,
    CONSTRAINT fk_livro_genero 
        FOREIGN KEY (genero_id) REFERENCES tb_generos(id)
        ON UPDATE CASCADE ON DELETE SET NULL
);
